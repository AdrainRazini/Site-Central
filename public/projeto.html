Projeto Em Grupo - 3º Ano B

Descrição do Projeto

Este projeto foi desenvolvido como parte do trabalho em grupo da turma 3º Ano B. O objetivo foi criar um site com funcionalidades interativas para mostrar as habilidades adquiridas ao longo do curso.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
  • Node.js

Objetivo do Projeto

Desenvolver um site interativo que possa ser utilizado como uma plataforma de aprendizagem online para diferentes públicos, mostrando projetos feitos pelos alunos e permitindo a interação entre eles.

Equipe

A equipe responsável pelo desenvolvimento deste projeto inclui os alunos Adrian Razini, João Silva e Ruan Gabriel.
